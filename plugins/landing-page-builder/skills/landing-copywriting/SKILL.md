# Landing Page Copywriting

Frameworks e templates para criar copy de alta conversao.

## Quick Start

1. Identifique a principal DOR do seu publico
2. Escolha um framework (PAS, PRESTO, AIDA)
3. Escreva o headline usando uma formula comprovada
4. Desenvolva cada secao seguindo o framework
5. Termine com CTA claro e urgente

## Frameworks de Copywriting

### PAS (Problem, Agitate, Solution)

*Problem*: Apresente a dor
```
Voce esta cansado de [problema]?
[Problema] esta te impedindo de [objetivo desejado]?
```

*Agitate*: Amplifique a dor
```
Imagine continuar assim por mais [tempo]...
Enquanto isso, [consequencia negativa]...
A cada dia que passa, [situacao piora]...
```

*Solution*: Apresente a solucao
```
Existe uma maneira melhor...
Apresentamos [solucao]...
Finalmente, uma forma de [resolver problema]...
```

### PRESTO Framework

| Etapa | Objetivo | Exemplo |
|-------|----------|---------|
| Pain | Identificar a dor | "Perder horas em tarefas repetitivas" |
| Resonate | Mostrar empatia | "Nos sabemos como e frustrante" |
| Educate | Explicar opcoes | "Existem 3 formas de resolver" |
| Simplify | Apresentar solucao simples | "Nossa ferramenta faz isso em 1 clique" |
| Testify | Prova social | "Mais de 10.000 usuarios satisfeitos" |
| Offer | Oferta irresistivel | "Teste gratis por 14 dias" |

### AIDA Framework

```
ATTENTION: Headline que para o scroll
INTEREST: Fato ou historia que gera curiosidade
DESIRE: Beneficios que criam vontade
ACTION: CTA claro e urgente
```

## Formulas de Headlines

### Headlines de Curiosidade
```
O segredo de [grupo de sucesso] para [resultado]
A verdade sobre [topico] que ninguem te conta
Por que [grupo] sempre consegue [resultado]
```

### Headlines de Beneficio
```
Como [resultado desejado] em [tempo] sem [objecao]
[Numero] maneiras de [resultado] garantido
Transforme [situacao atual] em [situacao desejada]
```

### Headlines de Urgencia
```
Ultima chance: [oferta] termina em [tempo]
Apenas [numero] vagas disponiveis para [beneficio]
[Beneficio] - mas apenas se voce agir agora
```

### Headlines de Prova Social
```
Descubra como [numero] pessoas ja [resultado]
O metodo usado por [autoridade] para [resultado]
Junte-se a [numero] que ja [conquistaram beneficio]
```

## Templates de Secoes

### Hero Section
```markdown
**Headline:** [Formula de headline escolhida]

**Subheadline:** [Expansao da promessa em 1-2 linhas]

**CTA:** [Verbo de acao] + [Beneficio] (ex: "Comece a economizar hoje")

**Prova rapida:** [Numero] usuarios | [Avaliacao] estrelas | [Garantia]
```

### Secao de Beneficios
```markdown
**Titulo:** O que voce vai conquistar

**Beneficio 1:** [Icone] [Titulo curto]
[Descricao de 1-2 linhas focada no resultado]

**Beneficio 2:** [Icone] [Titulo curto]
[Descricao de 1-2 linhas focada no resultado]

**Beneficio 3:** [Icone] [Titulo curto]
[Descricao de 1-2 linhas focada no resultado]
```

### Secao de Depoimentos
```markdown
**Titulo:** O que nossos clientes dizem

**Depoimento:**
"[Citacao especifica com resultado mensuravel]"
- [Nome], [Cargo/Empresa]
[Foto]
```

### CTA Final
```markdown
**Titulo:** [Pergunta retórica ou chamada urgente]

**Copy:** [2-3 linhas reforçando o principal benefício]

**CTA Principal:** [Botão com ação clara]

**Garantia:** [Selo de garantia ou promessa de risco zero]
```

## Palavras de Poder

### Urgencia
- Agora, Hoje, Imediatamente
- Limitado, Exclusivo, Ultimo
- Antes que acabe, Nao perca

### Beneficio
- Gratis, Garantido, Comprovado
- Facil, Rapido, Simples
- Novo, Revolucionario, Inedito

### Confianca
- Seguro, Protegido, Confiavel
- Certificado, Aprovado, Testado
- Recomendado, Premiado

## Checklist de Revisao

- [ ] Headline captura atencao em 3 segundos?
- [ ] Subheadline expande a promessa?
- [ ] Copy foca em beneficios (nao features)?
- [ ] Tem prova social especifica?
- [ ] CTA e claro e urgente?
- [ ] Linguagem e do publico-alvo?
- [ ] Objecoes foram tratadas?
- [ ] Garantia esta visivel?
