## Interatividade

O dashboard suporta:

- **Clique em lead**: Abre conversa completa
- **Responder**: Abre editor de mensagem
- **Ver historico**: Mostra timeline da conversa
- **Refresh**: Atualiza dados (auto a cada 30s)
- **Filtros**: Por score, tempo, classificacao

## Comandos do Dashboard

```
/wa-dashboard refresh      # Atualizar dados
/wa-dashboard fullscreen   # Modo tela cheia
/wa-dashboard export       # Exportar para PDF
/wa-dashboard alerts on    # Ativar sons de alerta
```
